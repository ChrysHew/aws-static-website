AWS Static Website

S3 + CloudFront Demo

Minimal static site deployed to a private S3 bucket and served globally via CloudFront (OAC). CI/CD-ready.

Stack

  • Amazon S3 (static hosting)
  • Amazon CloudFront (CDN + HTTPS)
  • Optional: GitHub Actions (OIDC)

Highlights

  • Public S3 website for demo, with clear path to private S3 + CloudFront OAC.
  • Region-aware setup (Montréal ca-central-1), easy to port.
  • Bucket policy + access-block configured intentionally for the mode.
  • Repo-ready structure (site/, docs/), simple to automate.
